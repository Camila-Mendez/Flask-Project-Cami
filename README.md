# 🇯🇵 Japanese Learning Web App (N5–N1)

Full-stack web application designed to support Japanese language learning from N5 to N1 levels, integrating user management, course tracking, and evaluation features.

---
## ⚠️ Note

This project uses an in-memory user system for demonstration purposes.  
No real database or persistent storage is currently implemented.

Passwords are securely hashed using Werkzeug.

Future versions will include database integration (e.g., SQL Server).

---

## 🌸Overview

This project is a web-based platform that allows users to study Japanese through structured levels (N5–N1). It provides tools for managing courses, tracking progress, and evaluating user performance.

The system was built to combine educational structure with scalable backend design, focusing on usability and data organization.

---

## 🚀 Technologies Used

- **Backend:** Python (Flask)
- **Frontend:** HTML, CSS, JavaScript
- **Database:** SQL Server (not yet connected)
- **Version Control:** Git & GitHub

---

- User registration and authentication

## ▶️ How to Run Locally

1. Clone the repository:
```bash
git clone https://github.com/your-username/japanese-learning-web-app.git
```

2. Navigate to the project folder:

```bash
cd japanese-learning-web-app
```

3. Install dependencies:
```bash
pip install -r requirements.txt
```

4.Run the application:
```bash
python app_cami.py
```
Open in browser:
```bash
http://localhost:5000
```
